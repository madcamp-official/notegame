========================================
  Ninja Adventure — 실행 방법
========================================

[준비물]
  1) KAIST VPN 접속  (필수! 없으면 게임 서버에 못 붙어요)

[실행]
  · Windows :  "게임 실행하기.bat"  더블클릭
  · Mac     :  "게임 실행하기.command"  더블클릭

  실행기가 서버로 가는 통로를 자동으로 열고 게임을 켭니다.
  게임을 끄면 통로도 자동으로 닫힙니다.

----------------------------------------
[잘 안 될 때]

· "게임 서버에 연결할 수 없습니다"
    -> KAIST VPN 이 켜져 있는지 확인하세요.

· (Mac) "확인되지 않은 개발자" 라며 안 열릴 때
    -> "게임 실행하기.command"를 마우스 우클릭 → "열기" → 다시 "열기".
       실행기가 게임 앱의 다운로드 격리 속성도 함께 정리합니다.

· (Windows) 파란 "Windows의 PC 보호" 창이 뜨면
    -> "추가 정보" → "실행" 을 누르세요.

· (Windows) ssh 를 찾을 수 없다고 나오면
    -> Windows 10 이상이면 기본 내장되어 있습니다.
       설정 → 앱 → 선택적 기능 에서 "OpenSSH 클라이언트" 설치.

----------------------------------------
[안내]
  · tunnel 폴더 안의 파일은 서버 접속용입니다. 지우지 마세요.
  · 이 통로는 게임 서버(하나의 포트)로만 연결되며,
    다른 곳에는 접근할 수 없도록 제한되어 있습니다.
